# Codeblocks demo

We use `<h1>` tag for heading in html. 

#  Codeblocks
```
<body>
    <h2>
    Hello World
    <h2>
</body>
```

# Language based codeblocks
```java
System.out.println("Hello World");
```
```html
<body>
    <div>
        <h1>Hello there</h1>
    </div>
</body>
````
```javascript
console.log("Hello js");
```
```c
printf("Hello World");
```
